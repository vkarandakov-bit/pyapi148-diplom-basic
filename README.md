# Курсовая работа «Ip Detector»

В Интернете часто ходит шутка, что можно вычислить по IP. Давайте реализуем эту шутку.

## Цель:
Разработать Python-программу, которая:

1. определяет текущий IP-адрес пользователя;
2. получает географическую информацию по этому IP через внешний API;
3. сохраняет полученные данные в формате JSON;
4. загружает файл на Яндекс.Диск через REST API.

## Задание:
1. С помощью сервиса ipify мы можем получить наш [ip адрес](https://api.ipify.org/?format=json)
2. Далее через сервис ipinfo по api можем получить город, где находится этот ip адрес `https://ipinfo.io/188.242.138.63/geo`
3. Необходимо сохранить эту информацию в json файл и загрузить на Яндекс.Диск
4. Информация по Яндекс Диску есть а  [Полигоне](https://yandex.ru/dev/disk/poligon/)
*Важно:* Токен Яндекс.Диска публиковать в github не нужно!

### Обязательные требования к программе:
1. Использовать REST API Я.Диска и ключ, полученный с полигона.
2. Для загруженного файла нужно создать свою папку на Я.Диске.
3. Загрузка файлов осуществляется по воздуху, т.е. локально фотографии не сохраняются. В Я.Полигоне есть необходимый метод.
4. Код программы должен удовлетворять PEP8.
5. У программы должен быть свой отдельный репозиторий. 
6. Все зависимости должны быть указаны в файле requiremеnts.txt.


### Критерии оценки:
1. Выполнены обязательные требования к программе.
2. Для каждого сервиса написан свой класс с методами.
3. Вызов методов у класса происходит в функции main.
4. Во время выполнения программы отсутствуют ошибки.
5. После выполнения программы не появляются дополнительные файлы на компьютере.
6. json-файл загружен на Я.Диск
